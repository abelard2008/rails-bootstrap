Rails Bootstrap + Sphinx
========================

Based on "Rails Bootstrap":https://github.com/RailsApps/rails-bootstrap, adding code from "Thinking Sphinx on Ruby on Rails: Part 1/2/3":http://sphinxsearch.com/blog/2015/01/06/thinking-sphinx-on-ruby-on-rails-part-3/

Rails 4.2.0
ruby 2.2.0

sphinx 2.0.8


commands:
* bundle install
* rake db:migrate
* rake ts:index
* rake ts:start
* rails s -p 9000
